# ICT-Loket Website

Responsive HTML/CSS website gebaseerd op de aangeleverde designs.

## Bestanden

- `index.html` — Hoofd HTML-structuur
- `styles.css` — Responsive styling
- `assets/logo.png` — Logo (plaats hier jouw logo)

## Hoe te gebruiken

1. Open `index.html` in een browser
2. Plaats je `logo.png` in de `assets/` map
3. Pas tekst en afbeeldingen naar wens aan

## Features

- Responsive design (desktop, tablet, mobiel)
- Hero sectie met teamfoto
- 3-koloms highlights grid
- Diensten sectie met 2-koloms layout
- Testimonials sectie
- Footer met contactinfo
